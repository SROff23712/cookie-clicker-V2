# Mini Cookie Clicker

## Lancer le jeu

- Ouvre `index.html` dans ton navigateur (double‑clic).

## Fonctionnalités

- Clic sur le cookie = cookies
- Boutique (coûts qui augmentent)
- Production automatique (cookies/seconde)
- Sauvegarde auto via `localStorage` + gain hors‑ligne

## Réinitialiser

- Bouton **Reset** en haut à droite
